## Usage
### Running the app in Development
  Start the development server
  ```
  npm run dev
  ```
  Open the app in your browser
  ```
  http://localhost:1337/
  ```

***********************************************

### Running the app in Production
  Build the app
  ```
  npm run build:prod
  ```
  Start the server
  ```
  npm run start:prod
  ```
